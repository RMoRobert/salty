How to Use This Tool

This tool helps you import recipes from websites into your Salty recipe library. You can extract text manually or use our automatic scanner (designed to work on most websites that use the JSON-LD Recipe schema.org format; select the the More (⋯) button, then select "Scan Webpage for Recipe Data").

Manual Text Extraction

For webpages that do not work with the scan tool, you can select text on the page and use these handy keyboard shortcuts to automatically copy them into the corresponding fields:

  • ⌘1 - Copy to Name field
  • ⌘2 - Copy to Source field
  • ⌘3 - Copy to Source Details field
  • ⌘4 - Copy to Servings field
  • ⌘5 - Copy to Ingredients text
  • ⌘6 - Copy to Directions text
  • ⌘7 - Copy to Yield field
  • ⌘8 - Copy to Introduction text

Text Cleanup

Tip: After importing ingredients or directions, use the "Clean Up Text" buttons to automatically clean up the formatting if you want to remove bullet points, numers, and other characters that you do not want in the actual ingredient or direction step text.
