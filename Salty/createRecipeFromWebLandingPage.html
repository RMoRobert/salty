Most websites that support the schema.org/Recipe standard in JSON-LD format should work with this tool, but compatibility with any individual site is subject to change at any time due to factors outside our control.

How to Use This Tool

This tool helps you import recipes from websites into your Salty recipe library. You can extract text manually (more easily with the shorcuts below!) or load any supported webpage and select the Auto-Import button.

Manual Text Extraction

(This option is not available on mobile devices.) For webpages that do not work with the scan tool, you can select text on the page and use these handy keyboard shortcuts to automatically copy them into the corresponding fields:

  • ⌘1 - Copy to Name field
  • ⌘2 - Copy to Source field
  • ⌘3 - Copy to Source Details field
  • ⌘4 - Copy to Servings field
  • ⌘5 - Copy to Ingredients text
  • ⌘6 - Copy to Directions text
  • ⌘7 - Copy to Yield field
  • ⌘8 - Copy to Introduction text

Text Cleanup

Tip: After importing ingredients or directions, use the Clean Up Text button to automatically clean up the formatting if you want to remove bullet points, numers, and other characters that may have been copied from the webpage that should not be part of the actual text.
